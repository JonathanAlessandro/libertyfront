# Liberty Saúde Landing Page

Estrutura separada em HTML, CSS, JavaScript e imagens.

```
liberty-saude-landing-organizada/
├── index.html
└── assets/
    ├── css/
    │   └── style.css
    ├── js/
    │   └── formulario.js
    └── images/
```

O formulário envia JSON para:

`https://api.libertysaude.com.br/api/clientes`

Campos enviados: `nome`, `email`, `telefone`, `cnpj`, `noCnpj`, `hospital`, `plano_atual`, `idades` e `aceitaEmails`.
